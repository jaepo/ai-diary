# AI 일기장 서비스 기획서

## 1. 서비스 개요

### 서비스명
AI 일기장 (AI Diary)

### 서비스 설명
사용자가 작성한 일기를 AI가 분석하여 감정 상태를 파악하고, 맞춤형 위로와 응원 메시지를 제공하는 웹 서비스

### 목표
- 사용자의 일기 작성 습관 형성
- AI를 통한 감정 관리 및 정서적 지지 제공
- 장기적 감정 패턴 인사이트 제공

---

## 2. 주요 기능 명세

### 2.1 사용자 인증
- **Google 소셜 로그인**
  - 간편한 가입/로그인
  - 별도 회원가입 절차 불필요
  - 보안성 확보

### 2.2 일기 작성
- **일기 에디터**
  - 텍스트 입력 (제한 없음)
  - 작성 날짜 자동 기록
  - 임시 저장 기능 (선택 사항)

### 2.3 AI 분석 (핵심 기능)
- **감정 분석**
  - 긍정적 (positive)
  - 부정적 (negative)
  - 중립적 (neutral)
  - 감정 점수: -1.0 ~ 1.0

- **AI 응답 메시지**
  - 일기 내용 기반 맞춤형 메시지
  - 공감적이고 따뜻한 톤
  - 2-3문장의 위로/응원

- **키워드 추출**
  - 일기의 핵심 주제 5개 추출
  - 해시태그 형태로 표시

### 2.4 일기 히스토리
- **목록 보기**
  - 최신순 정렬
  - 날짜별 표시
  - AI 분석 결과 요약 표시

## 4. 화면 구성

### 4.1 로그인 페이지
- Google 로그인 버튼
- 서비스 소개 문구
- 디자인: 심플하고 친근한 느낌

### 4.2 메인 페이지 (대시보드)
- **좌측**: 일기 작성 영역
  - 텍스트 에디터
  - 저장 버튼
  - AI 분석 결과 표시 영역

- **우측**: 일기 히스토리
  - 이전 일기 목록
  - 날짜/키워드/감정 표시
  - 클릭 시 상세 보기

- **상단**: 
  - 로고
  - 사용자 정보
  - 로그아웃 버튼

## 5. 데이터 구조

### User (사용자)
- id: 고유 ID
- email: 이메일
- name: 이름
- image: 프로필 이미지
- createdAt: 가입일

### DiaryEntry (일기)
- id: 고유 ID
- userId: 사용자 ID (외래키)
- content: 일기 내용
- sentiment: 감정 분류 (positive/negative/neutral)
- sentimentScore: 감정 점수 (-1.0 ~ 1.0)
- aiResponse: AI 응답 메시지
- keywords: 키워드 배열 (JSON)
- createdAt: 작성일
- updatedAt: 수정일
